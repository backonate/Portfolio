SFU Rocketry - Software Engineer

Sept 2023 - June 2024 | Vancouver, Canada

As a Software Engineer for SFU Rocketry, I designed and implemented a C++ module to automate the storage of rocket specifications in XML format. This automation reduced manual data entry errors by 30%, improving the reliability of our rocket documentation.

The project required a deep understanding of both software architecture and the specific needs of rocketry applications. I worked to create a robust system that could handle complex specification data while remaining user-friendly for team members.

Beyond the specification storage system, I built a physics-based rocket simulation in Unreal Engine. This simulator featured arrow key controls and realistic flight dynamics, providing the team with a valuable tool for visualizing rocket behavior.

This experience bridged my interests in mechanical systems and software development. Working on rocketry software taught me how to apply programming skills to solve real engineering challenges in an aerospace context.
